Setting it up

Prerequisites:
- BASH support, installed in /system/xbin
- Root Access (your device must be rooted)
- A custom rom with permission fixing (might be needed for sources that don't specify the right permission settings)
- A file manager to copy rpkg4android to /system/xbin, FX File Explorer is a good one. It is available from Google Play.
- Up-to-date busybox
- A terminal emulator, you can get one from the Play Store.

1. Installing RPKG4ANDROID

First, download a zipball or tarball by clicking one of the buttons above.
Then, unpack the archive to any directory you wish.
Only copy "firstrun", "rpkgupdate", "rpkginstall", "addsource" and "listsources" to /system/xbin. You may need to remount /system read-write.
Now, open a root shell and run "firstrun". It will not run properly. You'll need to change file permissions.
Run the following commands to fix file permissions:
chmod 777 /system/xbin/*

2. Add default repo

Run "firstrun" on your root shell. It should update and add the default repo.
That is pretty much it for the default repo.
